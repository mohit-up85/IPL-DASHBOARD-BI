# IPL-DASHBOARD-BI
📊 Project Title / Headline
IPL Data Analysis Dashboard (2008–2025) | Power BI Project

📝 Short Description / Purpose
This project focuses on analyzing Indian Premier League (IPL) data from 2008 to 2025 using Power BI.
The dashboard provides interactive insights into team performance, player statistics, match trends, and venue analysis.

The objective of this project is to transform raw IPL datasets into meaningful insights using data visualization and DAX measures.

⚙️ Tech Stack

Power BI – Dashboard Development
DAX (Data Analysis Expressions) – KPI calculations and measures
Power Query – Data cleaning and transformation
Data Modeling – Relationship creation between datasets
Excel / CSV – Data storage and preprocessing

📂 Data Source

The dataset used in this project includes IPL match and ball-by-ball data.

Main datasets used:
IPL Matches Dataset – Contains match details such as teams, venue, season, toss decision, and match results.
Ball-by-Ball Dataset – Contains delivery-level data including runs, wickets, batsman, bowler, and batting team.
Players Dataset – Contains player information.
Teams Dataset – Contains IPL team details.
Source: Public IPL datasets (commonly available on Kaggle and open cricket datasets).

⭐ Features / Highlights

✔ Interactive Power BI Dashboard
✔ Analysis of IPL matches from 2008–2025
✔ Key KPI metrics including:
Total Matches Played
Total Teams & Venues
Total Runs Scored
Total Sixes & Fours
Total Wickets
Average Runs per Match

✔ Orange Cap Holder Analysis – Highest run scorer
✔ Purple Cap Holder Analysis – Highest wicket taker

✔ Player Performance Insights:
Top Batsmen by Runs
Top Bowlers by Wickets

✔ Match Insights:
Toss Decision Analysis
Matches by City and Venue
Team Performance Overview

✔ Interactive filters for:
Season
Team
City
